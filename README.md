# Portfolio Website

Personal portfolio website showcasing data analytics, machine learning, Tableau, Power BI, and dashboard projects.

## Projects Included

- Predictive Data Analysis Project
- GDP Dashboard Project

## Tech Stack

- React
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
